# UI Guidelines

Use:

* React Native (Expo)
* React Navigation (Native Stack)
* TypeScript

Do not use Expo Router. See ADR 009.

UI Principles:

* minimal
* musician focused
* accessible

Performance:

Use:

React.memo
FlatList

Avoid:

Anonymous inline functions.

Accessibility:

Support:

* Dynamic fonts
* Screen readers
* Haptics
