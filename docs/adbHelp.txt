 On the OnePlus 6T, you can find Developer options here:

Open Settings.
Tap System.
Tap Developer options.

If you don't see Developer options, it means they're hidden. To enable them:

Open Settings.
Tap About phone.
Tap Build number 7 times.
Enter your PIN/password if prompted.

You should see a message like:

"You are now a developer!"

Then go back to:

Settings → System → Developer options

Inside Developer options, make sure:

✅ USB debugging is ON.
If available, set the Default USB configuration to File Transfer.
